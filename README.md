# 🚀 JavaScript Learning & Mastery

Welcome to my comprehensive JavaScript learning repository. This project is a microscopic, exhaustive deep dive into the core mechanics, advanced concepts, and enterprise-grade architecture of JavaScript.

The primary goal of this repository is to build an unshakable foundation for building scalable, high-performance web applications, particularly within modern tech stacks like MERN (MongoDB, Express, React, Node.js).

## 🛠️ Tech Stack & Environment

- **Core Language:** JavaScript (ES6+)
- **Runtime Environment:** Node.js
- **Recommended IDE:** Visual Studio Code

## 🏗️ Architectural Philosophy

To maintain clarity and ensure highly focused learning, this repository strictly adheres to the **Single Responsibility Principle (SRP) at the file level** across all phases and modules.

Every single `.js` file is dedicated to demonstrating and explaining _one_ specific concept, engine mechanic, or data structure. This prevents cognitive overload and makes the repository highly searchable as a reference guide.

## 🚀 Getting Started

### Prerequisites

To run these files locally, you will need the following installed on your machine:

- [Node.js](https://nodejs.org/) (Required for the runtime environment)
- [Visual Studio Code](https://code.visualstudio.com/) (Recommended IDE, but optional)

### Clone the repository:

```bash
git clone https://github.com/anandprakash01/js-learning.git
```

### Execution

Since every file operates independently under the Single Responsibility Principle, you can execute any script directly to see the output and behaviors in real-time. Choose the method that best fits your workflow:

#### Option 1: Terminal / Node CLI (Standard)

```bash
# Navigate to the specific module
cd 04_engine_mechanics

# Run the target file
node 03_execution_context.js
```

#### Option 2: Visual Studio Code

If you are using VS Code, you can open the project and run the files directly using any of the following streamlined options:

A. Integrated Terminal

Open the terminal and run the commands exactly like the standard CLI:

```bash
# Open VS Code Integrated Terminal and navigate to the module
cd 04_engine_mechanics

# Run the target file
node 03_execution_context.js
```

B. Code Runner Extension (Fastest)

1. Install the Code Runner extension from the VS Code Marketplace.
2. Open any .js file in the editor.
3. Press Ctrl + Alt + N (Cmd + Option + N on Mac) or click the "Play" ▶️ button in the top right corner to instantly execute the script and view the output.
